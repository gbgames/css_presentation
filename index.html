CSS: The Interior Decorator

If HTML is marked-up content, then CSS is the presentation of that content.

CSS styles allow you to bring out the meaning of HTML. For example:

I used <h1> here to say this is a heading

is styled with:

h1 { font-family: serif; font-weight: bold; font-size: 46px; }

It could just as easily have been styled with:

h1 { font-family: sans-serif; font-weight: lighter; font-size: 12px; color: red; }

And it would look like this, which isn't very heading-ish, is it?

The Box Model

The margin surrounds the box. It says "Don't let other boxes come too close to me"

Margins don't add up between boxes. They "collapse" instead.

I imagine not knowing about the collapse can result in a Peter-Griffin-wrestling-with-blinds situation.
